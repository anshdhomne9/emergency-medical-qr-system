🚑 Emergency Medical QR System

A web-based system to store and access medical data using QR codes.

🔥 Features
- User Login & Registration
- Medical Data Storage
- QR Code Generation
- QR Scanner Simulation
- Modern UI (Glassmorphism)

🛠 Tech Stack
- HTML
- CSS
- JavaScript

 🚀 How It Works
1. User enters medical details
2. System stores data using localStorage
3. QR code is generated
4. Scanning QR opens user data page

 📌 Future Improvements
- Backend integration (Node.js)
- Real QR Scanner
- Database (MongoDB)
